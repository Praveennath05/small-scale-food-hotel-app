# Small Scale Food Hotel Web App
A Full Stack MERN (MongoDB, Express.js, React.js, Node.js) web application built to help small-scale food hotels manage products, billing, payments, and sales records efficiently.

### User Login
- Secure user authentication
- Ensures data privacy and control

###  Product Management
- Add new food products
- Edit product details
- Update product price and availability
- Delete product

### Billing System
- Create customer bills dynamically
- Automatic total calculation
- Supports multiple items per bill

### Payment System
- Handles digital and manual payments
- QR-based payment workflow support

### Selling History
- Stores daily and past sales records
- Useful for basic reporting and analysis

## Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Bootstrap

### Backend
- Node.js
- Express.js

### Database
- MongoDB
